# 贴敷式胰岛素泵 — 产品文件全生命周期清单

> **产品描述**：贴敷式胰岛素泵（Patch Insulin Pump）
> **法规分类**：有源医疗器械 + 输注泵 + 含软件组件 + 无菌提供 + 贴敷式
> **管理类别**：III 类医疗器械（NMPA）
> **适用标准体系**：ISO 13485, ISO 14971, IEC 62304, IEC 62366, GB 9706.1, GB 9706.224, ISO 10993, ISO 11607

```

├── 检验方法（检验SOP、接收标准）
└── 包装和标签规范
```

```

├── 批检验记录
├── 灭菌过程记录
├── UDI赋码记录
└── 放行记录
```
