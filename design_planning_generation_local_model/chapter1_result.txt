### 1. 产品概述  

| 项目 | 内容 |
|------|------|
| **产品名称** | 测试血糖仪（型号：BG‑M‑01） |
| **产品分类** | 有源医疗器械（Active Medical Device） |
| **预期用途** | 用于定量测量人体毛细血管血样中的葡萄糖浓度，帮助糖尿病患者或其护理人员自行监测血糖水平，以便进行疾病管理和治疗调整。 |
| **测量范围** | 1.1 mmol/L – 33.3 mmol/L |
| **核心技术** | 电化学传感技术（酶电极法）+ 自动校准算法 |
| **主要组成部件** | 主机、手柄（或探头）采样口、显示/操作界面、电池/电源管理模块、通信接口（蓝牙/USB） |
| **适用范围** | 适用于医院门诊、社区卫生站以及家庭自我检测；适用于成人及≥7 岁儿童（需在成人监督下使用）。 |

---

### 2. 预期使用环境与使用者  

1. **使用环境**  
   - 清洁、干燥的室内环境；相对湿度30 % – 70 %；温度 10 °C – 35 °C。  
   - 不适用于高压、快速血流或极端环境（如强磁场、强光直射）等特殊场景。  

2. **目标使用者**  
   - 患者本人（具备基本操作能力）。  
   - 医护人员（负责培训、校准及结果判读）。  

3. **使用限制**  
   - 不适用于血糖浓度超出 1.1 mmol/L – 33.3 mmol/L 范围的血样（超出范围提示“超出测量范围”）。  
   - 采样方式仅限于指尖毛细血管血样或其他经临床验证的血样类型（如耳垂血），不适用于动/静脉血样的直接测量。

### 4. 风险管理过程概述  

本血糖仪的风险管理按照 **ISO 14971:2019** 的要求开展，主要包括以下阶段：

| 阶段 | 关键活动 | 输出文件 |
|------|----------|----------|
| **风险管理计划（RMP）** | - 确定适用范围、风险可接受准则 <br>- 分配职责、进度、资源 <br>- 规定评审与批准程序 | 风险管理计划（文件编号：RMP‑BG01） |
| **危害识别（ Hazards Identification）** | - 列出可能的危害（测量误差、电气安全、软件失效、使用错误等）<br>- 分析危害情境（误操作、校准失误、环境干扰等） | 危害清单（Hazard List） |
| **风险评估（Risk Estimation & Evaluation）** | - 估计每种危害的发生概率、严重程度<br>- 计算风险值（PV = Prob × Sev）<br>- 与可接受风险准则比较 | 风险评估矩阵（Risk Matrix） |
| **风险控制（Risk Control）** | - 设计防护措施（如软硬件容错、双重校准、用户提示）<br>- 采用最优化原则（ALARP）<br>- 验证控制措施有效性 | 风险控制措施（Risk Control Measures） |
| **残余风险评价** | - 评估实施控制后残余风险是否可接受<br>- 如不可接受，重新设计或改进 | 残余风险报告 |
| **风险管理报告（RM Report）** | - 汇总全过程、结论、建议、批准签字 | 风险管理报告（RM‑BG01） |
| **评审与批准** | - 由风险管理小组（包括ISO 14971 内审员）进行评审<br>- 由质量管理负责人批准 | 批准记录 |

> **注**：所有过程均记录在电子文档管理系统（EDMS），确保可追溯性和版本控制。

### 5. 风险管理组织结构与职责  

| 角色 | 主要职责 |
|------|----------|
| **风险管理责任人（RMP Owner）** | 负责制定并维护风险管理计划，监督各阶段的执行情况，组织评审会议。 |
| **危害分析工程师** | 主导危害识别、危害情境分析，输出危害清单与危害报告。 |
| **风险评估工程师** | 进行风险概率/严重度估计，构建风险评估矩阵，提供可接受风险准则的建议。 |
| **风险控制工程师** | 设计并验证风险控制措施，跟踪实施效果，确保满足 IEC 62304、IEC 60601‑1 等相关标准。 |
| **ISO 14971 内审员** | 对风险管理过程进行内部审计，确保符合 ISO 14971:2019 要求，起草审计报告并推动改进。 |
| **质量管理部（QMU）** | 负责批准风险管理报告，确保文档符合公司质量管理体系（QMS）。 |
| **临床/技术支持部** | 提供临床使用反馈、上市后警戒信息，协助风险控制措施的验证。 |

所有角色的任职资格均经公司人力资源部确认，风险管理团队成员具备 **ISO 14971 内审员** 或同等能力的培训记录。

### 7. 与其他质量管理过程的关联  

1. **设计与开发过程（Design & Development）**  
   - 风险管理活动在设计阶段即开始嵌入，确保危害分析结果能够直接反馈到设计规格书中。  
   - 设计变更需经风险再评估后方可实施。  

2. **生产与质量控制（Production & Quality Control）**  
   - 生产过程的关键工序（如校准、贴标）列入风险管理控制点。  
   - 每批出厂产品需进行 **校准验证** 与 **功能检查**，并记录在批次检验报告中。  

3. **临床评价（Clinical Evaluation）**  
   - 临床数据的收集和分析用于验证血糖测量误差是否在可接受范围内（即符合 ISO 15197 标准）。  
   - 临床随访结果将作为 **上市后警戒** 输入，定期更新风险管理报告。  

4. **上市后监督（PMS）**  
   - 建立 **不良事件报告系统**（AERS），对任何测量误差、误操作等异常进行收集、评估。  
   - PMS 数据用于持续改进风险控制措施，形成 **闭环**。  

5. **内部审计**  
   - ISO 14971 内审员每年对风险管理过程进行一次内部审计，审计结果直接影响质量管理体系的持续改进。
