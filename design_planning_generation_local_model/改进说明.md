# 系统改进说明

## 1. 模型更新
- 将模型从 `glm-5.1` 更新为 `Doubao-Seed-2.0-pro`
- API Key 保持不变

## 2. 生成内容优化

### 2.1 Prompt 大幅增强
- 所有文档类型的提示词都进行了扩展和细化
- 增加了10项详细的生成要求
- SOP（作业指导书）特别强调可操作性、详细步骤、质量控制点
- 强调使用专业的医疗器械行业术语

### 2.2 API 参数优化
- `temperature`: 0.4 → 0.7（提高内容丰富度和创造性）
- `max_tokens`: 8000 → 12000（支持更详细的内容输出）

### 2.3 RAG 检索优化
- `top_k`: 5 → 8（获取更多参考内容）
- `similarity_threshold`: 0.3 → 0.25（降低阈值，获得更多参考）
- RAG 提示词也进行了增强，强调详细程度要求

## 3. 网络搜索增强

### 3.1 搜索源改进
- 主搜索从 DuckDuckGo 切换到 Bing（中文结果更好）
- DuckDuckGo 作为备用搜索源
- 增加更多 SOP 专用搜索查询

### 3.2 深度网页抓取
- 自动抓取搜索结果页面的详细内容（不仅是摘要）
- 支持多种内容选择器（main, article, div.content, 等）
- 抓取内容限制在 4000 字符以内

### 3.3 文件自动下载
- 自动检测并下载 PDF/DOC/DOCX 等文档文件
- 下载的文件保存到 `downloads/web/{doc_type}/` 目录
- 支持的文件类型：.pdf, .doc, .docx, .txt, .xls, .xlsx

## 4. 知识库增强

### 4.1 多格式支持
- **.docx**: 完整支持（标题、段落、表格）
- **.pdf**: 支持 PyPDF2 和 pdfplumber（推荐）
- **.doc**: 支持 textract（需要额外安装）
- **.txt**: 支持多种编码（UTF-8, GBK, GB18030）

### 4.2 自动知识库更新
- 下载的文件自动添加到向量知识库
- 自动识别或强制指定文档类型
- 支持实时更新，下次生成即可使用新内容

### 4.3 Chunk 配置优化
- `MAX_CHUNKS_PER_DOC`: 30 → 50（保留更多内容）
- `CHUNK_SIZE`: 1000 → 1200（更完整的语义单元）
- `CHUNK_OVERLAP`: 100 → 150（更好的连贯性）

## 5. 新增便捷脚本

### 5.1 `ingest_sop.py`
- 专门用于摄入作业指导书到知识库
- 自动扫描多个目录
- 支持交互式确认

### 5.2 `restart_service.py`
- 一键重启服务
- 自动清理旧进程
- 管理服务生命周期

## 6. 使用方法

### 6.1 初始化知识库（推荐首次使用）
```bash
# 激活环境
conda activate env_01

# 摄入作业指导书目录
python ingest_sop.py

# 或摄入所有文档
python -m app.services.rag.ingest --rebuild
```

### 6.2 重启服务
```bash
# 使用便捷脚本
python restart_service.py
```

### 6.3 生成文档
- 访问 http://localhost:8001
- 生成时会自动：
  - 搜索相关网页内容
  - 下载相关文档
  - 更新知识库
  - 生成更详细的内容

## 7. 推荐安装的依赖

为了获得最佳体验，建议安装：

```bash
# PDF 解析（推荐 pdfplumber）
pip install pdfplumber PyPDF2

# DOC 解析（可选）
pip install textract

# 其他依赖（应该已安装）
pip install python-docx chromadb sentence-transformers
```
